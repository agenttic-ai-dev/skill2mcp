# Security Policy

## Supported versions

Security fixes are provided for the latest released minor line.

## Reporting a vulnerability

Please report vulnerabilities privately through GitHub Security Advisories:

- `Security` tab -> `Report a vulnerability`

Do not open public issues for security vulnerabilities.

## Response targets

- Acknowledgment: within 72 hours
- Triage and severity classification: within 7 days
- Patch timeline: depends on severity and blast radius

## Disclosure

We follow coordinated disclosure. A fix and advisory will be published after remediation.
